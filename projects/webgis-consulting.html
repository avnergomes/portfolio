WebGIS Platform Consulting

User experience evaluation and product improvement consulting for a satellite imagery startup, providing actionable feedback from a real user perspective.

Geospatial
2024
Cambodia / Remote

Project Overview

Engaged by a satellite imagery startup to provide comprehensive platform evaluation from an end-user perspective. The project involved using their WebGIS platform as a real customer would, testing workflows, identifying pain points, and proposing improvements.

The assessment covered user interface design, data processing workflows, export functionality, and overall user experience, resulting in a prioritized list of actionable recommendations.

Delivered prioritized list of UX improvements and feature ideas, several of which were implemented in the next platform release.
