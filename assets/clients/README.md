# Client logos

Drop each official logo here using the exact filename below. The marquee picks
it up automatically (no code change). If a file is missing, that client shows a
text wordmark instead — no broken images.

## Format
- **Transparent background** (SVG preferred, or transparent PNG). The site
  renders every logo as a uniform **white silhouette** on the dark wall and
  reveals full colour on hover, so a transparent mono/flat logo works best.
- Accepted extensions, tried in order: `.svg` → `.png`.
- Aim for ~height 200px+ (PNG) or any size (SVG). Wide wordmark logos look best.
